## Database 

### Datatypes

* BOOLEAN

Following are some valid literals for boolean values

| TRUE        | FALSE       |
| ----------- | ----------- |
| TRUE        | FALSE       |
| 'true'      | 'false'     |
| 't'         | 'f'         |
| 'y'         | 'n'         |
| 'yes'       | 'no'        |
| '1'         | '0'         |

* CHARACTERS


| CHARACTER TYPES           | NOTES                        |
| ------------------------- | ---------------------------- |
| CHARACTER(n), CHAR(n)     | fixed-length, blank padded   |
| VARCHAR(n)                | var length with length limit |
| TEXT, VARCHAR             | var unlimited length         |
